Speed!


Details

  • Technologies Used: ASP.NET, SignalR, Angular, Visual Studio 2022 (IDE)
  • Group assignment in which a player plays the original Speed card game.
  • The game was hosted locally.
  • The web-based card game itself was created using asynchronous UI with the help of SignalR which allowed asynchronous communication between two players and the server.
  • Authentication was not required. Players were able to play from different sessions (two different browsers on the same computer, or two different computers).
  • The game used websockets to transmit data back and forth via local server.
  • Logic of game was generated server-side.
  • Contribution: Design of the contents of the game by the use of styles, collaborate with my teammates to implement the logic of the game, implement many SignalR handlers, add server-side functions that handle the reset and the creation of the game, assist my teammates with the drop card mechanism.
