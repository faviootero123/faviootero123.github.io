Hangman!


Details

  • Technologies Used: ASP.NET, Blazor, EF Core, Visual Studio 2022 (IDE)
  • Group assignment in which a player plays the traditional Hangman game.
  • The game was hosted locally.
  • The game itself was created using asynchronous UI (no redirection) with the help of Websockets which allowed true asynchronous communication between server and client.
  • The game was also able to display a list with the highest score from the top players.
  • Account authentication was required. My team decided to use the built in authentication from Blazor.
  • Logic of game was generated server-side.
  • Contribution: Design of the contents of the game by the use of styles, collaborate with my teammates to implement the logic of the game, assist my teammates with the database migrations.
